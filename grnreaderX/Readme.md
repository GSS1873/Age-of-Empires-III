New features from the grnreader
----
- importing bones
- vertex weighing
- loading animations

Requirements
----
- 3ds Max 7 or later
- AoE3ED to extract .gr2 files from game archives

How to use
----
drop a .gr2 file directly on grnreaderX.exe or execute it from cmd:
> grnreaderX model.gr2 animation.gr2
